Timestamp Microservice

Powered by NodeJS, Express and Glitch

Instructions

Enter a timestamp in the url to use the timestamp microservice.

Example Usage

https://stitch-deltadromeus.glitch.me/api/timestamp/2019-01-12

https://stitch-deltadromeus.glitch.me/api/timestamp/1570486448

Example Output

{"natural time":"January 12th 2019","unix time":"1547251200"}
